# Ansor Med

Корпоративный сайт поставщика медицинского оборудования **Ansor Med**. Стиль — «Luminous Clinical»: светлая премиальная тема, точная типографика IBM Plex и **реальная 3D-модель нейросети** (React Three Fiber) в герое, оформленная как «приборное окно» с калибровочными скобками и mono-надписями.

## Стек

- **Next.js 16** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS v4** (дизайн-токены в `app/globals.css`)
- **React Three Fiber** + **drei** + **postprocessing** — 3D-модель нейросети (`components/three/`)
- **Framer Motion** — появления при скролле, переходы между страницами, микроанимации
- **lucide-react** — SVG-иконки
- Шрифты: **IBM Plex Sans** + **IBM Plex Mono**

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # продакшн-сборка
npm start        # запуск собранной версии
```

## Структура

```
app/
  page.tsx                 Главная (герой, преимущества, категории, статистика, блог, CTA)
  products/                Каталог: живой поиск, фильтр-чипы, сортировка
  products/[slug]/         Карточка товара: характеристики, похожее
  blog/  blog/[slug]/      Блог и страница статьи
  about/                   О компании: миссия, ценности, история, партнёры
  contacts/                Контакты: форма с валидацией + карта
  template.tsx             Анимированный переход между страницами
components/                Header, Footer, секции, карточки, motion-обёртки, форма
lib/data/                  Контент (товары, статьи, категории, тексты) — легко редактировать
```

## Как редактировать контент

Весь демо-контент вынесен в `lib/data/`:

- `products.ts` — товары (12 позиций) и характеристики
- `posts.ts` — статьи блога
- `categories.ts` — категории каталога
- `site.ts` — контакты, преимущества, статистика, ценности, история, партнёры

## Доступность и производительность

- Учитывается `prefers-reduced-motion` (анимации и нейросеть замедляются/останавливаются)
- Видимые фокус-кольца, skip-link, aria-метки, семантическая разметка
- Адаптив: 375 / 768 / 1024 / 1440 px
- Изображения-визуалы генерируются на лету (SVG), без внешних зависимостей
- 3D загружается через `next/dynamic` (`ssr:false`); при `prefers-reduced-motion` модель статична

> Контактная форма использует имитацию отправки (`setTimeout`). Подключите реальный
> API в `components/forms/ContactForm.tsx` (функция `onSubmit`).
